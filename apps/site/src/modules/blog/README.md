# Blog Module (Public)

Este modulo concentra o dominio e a interface publica do blog.

Escopo planejado:
- listagem publica de posts (`/blog`)
- pagina de detalhe de post (`/blog/[postId]`)
- contratos de leitura separados do CMS (`blog-admin`)

Diretriz:
- leitura publica em `src/modules/blog/*`
- escrita/admin em `src/modules/blog-admin/*`
